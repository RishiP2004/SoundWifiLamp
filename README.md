# SoundWifiLamp
A project to create a smart lamp controlled by either a sound sensor or app buttons.
This project integrates sound detection and wifi technology, using Arduino and ESP32.

##Main Components:

- Arduino Uno
- ESP32
- Sound Sensor
- ?kOhm Resistor
