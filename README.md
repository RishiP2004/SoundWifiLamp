# SoundWifiLamp
A project to create a smart lamp controlled by either a sound sensor or app buttons using website via WiFi.
This project integrates sound detection and wifi technology, using ESP32-WROOM-32D.

##Main Components:

- ESP32-WROOM-32D
- Sound Sensor KY-038
- 180 Ohm Resistor
- Green LED
- HW-479 RGB Module
- 2 x orange wire
- 5 x blue wire
- 4 x black wire
